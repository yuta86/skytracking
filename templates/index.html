Необходимо написать WEB API на основе фреймвёрка flask, celery и sqlalchemy.


API имеет 5 функций:

Умножение - метод POST, на вход подается 2 параметра, на выходе результат в формате JSON

Деление - метод POST, на вход подается 2 параметра, на выходе результат(float) в формате JSON

Сложение - метод POST, на вход подается 2 параметра, на выходе результат в формате JSON

Возведение в степень - метод POST, на вход подается 2 параметра, на выходе результат в формате JSON

Отчет о работе api за последние сутки - метод GET, на вход подается 0 параметров, на выходе результат

суммарного вызова каждой математической функции в формате JSON вида:

{'Сложение': 5, 'Умножение': 6, 'Деление':10, 'Возведение в степень':123}


На каждый вызов математических функций должна быть запись в таблице логов в БД.

Лог должен содержать следующую информацию (минимум):

Дата

IP-адрес клиента

Входные параметры

Результат работы функции


API должно являться проксирущей прослойкой для данных. Все вычисления реализовать через библиотеку celery (в процессе разработки рекомендую использовать флаг task_always_eager=True в настройках celery) Вся работа с базой данных ведется через ORM SqlAlchemy Выбор базы данных может быть любой, но в рамках тестового задания есть смысл ограничиться sqlite
